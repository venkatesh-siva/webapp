# webapp

## Prerequisites

    GIT
    JDK 8 or later
    Maven 3.0 or later

## Clone

### To get started you can simply clone this repository using git:

    git@github.com:sivakumarve-fall2020/webapp.git

### Build the project using the following command:
    
    mvn clean install

### Run the application using the following command:

    Cd csye

    mvn spring-boot:run
